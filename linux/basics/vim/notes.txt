Редактирование файла с начала  - I
C предыдущей позиции - i 
C конца строки - А
C новой строки ниже - o
C новой строки выше - O
s - замена текущего символа
S - замена текущей строки
r - заменить текущий символ
R - заменить несколько последовательно расположенных символов
h - влево
j - ввниз
k - вверх
l - вправо
0 - начало строки
^ - начало строки
$ - конец строки
ctrl + b - перемещение на экран назад
ctrl - f - перемещение на экран вперёд
Number + G - переход по абсолютному номеру строки
H - первая строка в окне
M - средня строка в окне
L - нижняя строка в окне
w - переход на слово вперёд
b - переход на слово назад
( - переход на предложение назад
) - переход на предложение вперёд
{ - переход на абзац назад
} - переход на абзац вперёд
mX - создать закладку
'X - перейти на закладку
ctrl + o - переход назад
ctrl + i - переход вперёд
v - visual mode
ap - выделить параграф
aw - выделить слово
ab - выделить блок со скобками
d(перемещение/выделение) - удаление
dd - удалить строку
y(перемещение/выделение) - копирование
yy - скопировать строку
x - удалить один символ в текущей позиции курсора.
p - вставить после текущей позиции курсора.
:e! - отмена всех изменений
